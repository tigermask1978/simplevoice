Free Speech Recognition Tool

This is a completely free and open-source speech recognition application that helps users convert speech to text quickly and privately.

Key Features

  • Real-time speech-to-text conversion
  • Supports Chinese and English (and more languages)
  • Local model running — no internet required, privacy protected
  • Simple and user-friendly interface
  • Fully open source — free to use, modify, and contribute
